# EulerStream Python SDK

This is an API wrapper for the Euler Stream API written in Python.
With this API you can access any Euler Stream public endpoint.

## Getting Started

Getting started is super simple! Remember to check out the OpenAPI Spec on the Euler Stream website
at https://www.eulerstream.com/docs/openapi.
